## 📊 DIE SITUATION

### **Was passiert ist:**

1. **September 2021:** Sinch kauft MessengerPeople für €48 Millionen
   - €33.6 Millionen in Cash
   - €14.4 Millionen in Sinch-Aktien
   - MessengerPeople wird Teil von Sinch AB

2. **Strategie:** Konsolidierung der Produkte
   - MessengerPeople: SaaS-Plattform für Mid-Market
   - Sinch Conversation API: Enterprise-API
   - Beide ergänzen sich, aber werden konsolidiert

3. **2026:** MessengerPeople API wird eingestellt
   - `messengerpeople.dev` API endet am **31.12.2026**
   - Neue Channels nur bis **31.01.2026**
   - Migration zu Sinch Conversation API empfohlen
