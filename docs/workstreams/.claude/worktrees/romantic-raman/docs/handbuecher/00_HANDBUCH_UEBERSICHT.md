## Quick-Start für alle Mitarbeiter

### Login
1. Browser öffnen: **https://drive.auto-greiner.de**
2. Mit Windows-Anmeldedaten einloggen (gleiche wie PC-Anmeldung)
3. Nach erfolgreichem Login erscheint das Dashboard

### Navigation
- **Linke Sidebar:** Hauptmenü mit allen freigeschalteten Modulen
- **Oben rechts:** Benutzername, Abmelden, Einstellungen
- **Dashboard:** Startseite mit wichtigsten KPIs

### Hilfe & Support
- Bei Problemen: IT-Abteilung kontaktieren
- Feature-Wünsche: An Florian Greiner melden
