## WICHTIG: Immer AD-Adressen verwenden!

E-Mail-Adressen in Scripts MÜSSEN aus dem Active Directory stammen.
Die Adressen enthalten oft **Umlaute** (ö, ü, ä)!

### Falsch:
```python
"m.koenig@auto-greiner.de"      # FALSCH - geraten
"f.greiner@auto-greiner.de"     # FALSCH - geraten
```

### Richtig:
```python
"matthias.könig@auto-greiner.de"  # RICHTIG - aus AD
"florian.greiner@auto-greiner.de" # RICHTIG - aus AD
```

## So findest du E-Mail-Adressen

### Option 1: Aus Datenbank
```sql
SELECT first_name, last_name, email
FROM employees
WHERE email IS NOT NULL
ORDER BY last_name;
```

### Option 2: Python
```python
from api.db_connection import get_db
conn = get_db()
cur = conn.cursor()
cur.execute("SELECT first_name, last_name, email FROM employees WHERE last_name ILIKE '%könig%'")
print(cur.fetchall())
```

## Hinweise zu Umlauten

- Die E-Mail-Adressen im AD enthalten echte Umlaute: ö, ü, ä, ß
- Microsoft 365 / Exchange unterstützt das
- In Python UTF-8 Encoding verwenden
- Bei Problemen: Umlaut-Variante UND ASCII-Variante testen
