## 🎓 LEARNINGS

### 1. Flask 308 Redirect-Handling
```
/bankenspiegel → [HTTP 308] → /bankenspiegel/ → [HTTP 302] → /dashboard

Flask fügt automatisch trailing slash hinzu (308).
Daher müssen beide Varianten abgefangen werden:
- @route('/') 
- @route('')
```

### 2. LDAP SSL mit selbst-signierten Zertifikaten
```python
# Für interne AD-Server ohne vertrauenswürdige Zerts:
tls_config = Tls(
    validate=ssl.CERT_NONE,        # Keine Validierung
    version=ssl.PROTOCOL_TLS,
    ciphers='DEFAULT@SECLEVEL=1'   # Relaxed für ältere Server
)
```

### 3. Multi-Rollen-System (Medium Complexity)
```
User ↔ Roles: Many-to-Many
AD-Groups ↔ Roles: Many-to-Many (auto-assignment)
Permissions: JSON in roles.permissions_json
```
