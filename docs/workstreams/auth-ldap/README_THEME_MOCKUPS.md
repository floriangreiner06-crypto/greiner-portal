# DRIVE – Drei Vorschlags-Mockups für zentrales Theme

Drei eigenständige HTML-Mockups zum Vergleichen. Im Browser öffnen (Dateiname doppelklicken oder `file:///...`).

| Datei | Name | Kurzbeschreibung |
|-------|------|------------------|
| **MOCKUP_THEME_1_DEEP_BLUE.html** | Deep Blue | Dunkelblaue Navbar + Hero (Verlauf #1e3a5f), einheitlicher Akzent für Badges/Buttons. Schrift: **DM Sans**. Ruhig, vertrauenswürdig. |
| **MOCKUP_THEME_2_SLATE_TEAL.html** | Slate & Teal | Graue Navbar (#334155), **Teal** (#0d9488) als Akzent für Icons, Buttons, Hover. Schrift: **Inter**. Sachlich, modern. |
| **MOCKUP_THEME_3_LIGHT_GREEN.html** | Light & Green | **Weiße Navbar** mit Grün (#166534) nur für Logo und Akzente. Heller Hero. Schrift: **Plus Jakarta Sans**. Frisch, „Autohaus“-Grün. |

Alle drei: gleicher Aufbau (Navbar, Hero, Quickinfo-Badges, Modul-Karten, Primär-Button), nur Farben und Schriften variieren. Keine Verspieltheit, professionell.

**Nächster Schritt:** Gewähltes Theme als `drive-theme.css` (CSS Custom Properties + Bootstrap-Overrides) umsetzen und in `base.html` einbinden.
